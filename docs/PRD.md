# EnglishWorld — 产品需求文档

> 目标用户：上海 12 岁男生（某人的儿子），初一/初二，有手机或平板，讨厌枯燥刷题
> 核心命题：**让语法和词汇练习像养宠物一样上瘾，而不是像做黄冈题库一样痛苦**

---

## 一、产品定位

### 一句话描述
> "一个做题赚金币养宠物、同时搞定语法和词汇的英语练习 App，只有你家孩子一个人用。"

### 核心原则

1. **5 秒规则** —— 任何页面如果 5 秒内没有让小朋友觉得"有点意思"，他就滑走了
2. **无痛学习** —— 把语法藏在玩法后面，不要让他觉得自己在做题
3. **即时反馈** —— 每个操作立刻有结果（对/错/金币/音效/动画）
4. **碎片化** —— 一局不超过 3-5 分钟，课间/蹲坑/等车都能来一局
5. **宠物驱动** —— 做题赚金币 → 金币养宠物 → 宠物开心了 → 还想再赚金币

## 二、用户画像

### 他（你家孩子）

| 属性 | 描述 |
|---|---|
| 年龄 | 12 岁，上海初一 |
| 英语现状 | 基础尚可，语法一团浆糊。知道"是这么说的"，但不知道为什么 |
| 痛点 | 老师讲语法太无聊，做题像坐牢。妈妈买的教辅连翻都不想翻 |
| 娱乐偏好 | 打游戏、刷 B 站、刷短视频 |
| 学习习惯 | 不做预习，不复习 |
| 最大动力 | 养宠物、看宠物进化、解锁新东西 |
| 最大阻力 | 无聊。只要无聊，立刻放弃 |

### 一句话总结

> 他不会因为"为了你好"而打开 App。他打开它，是因为**想喂他的小恐龙**。

## 三、核心机制

### 3.1 世界观

**"语法与词汇世界"**

整个英语世界分为四大区域：

- **🗡️ 语法大陆** — 时态、语态、从句、介词……每个知识点是一座城堡/Boss
- **📖 词汇森林** — 中考必背 1600+ 词，按主题分成不同的"树林区域"
- **📚 阅读高塔** — 层层向上的高塔，每层是一篇阅读理解文章
- **📻 电波台** — 听力专区，复古收音机界面，收听英文录音做题

四个区域相辅相成：词汇是基础，语法是规则，阅读是综合，听力是实战。

小朋友扮演勇士，通过答题闯关收集金币。金币用来**喂养和进化他的宠物**。

```
答题赚金币 → 养宠物 → 宠物进化 → 解锁新地图 → 答更多题 → 赚更多金币
                     ↑______________________________↓
```

### 3.2 驱动力设计（单人版）

| 驱动力 | 实现方式 |
|---|---|
| 好奇心 | 宠物下一阶段长什么样？新地图里有什么？ |
| 收集欲 | 勋章墙、宠物皮肤、称号 |
| 进步感 | 经验条 + 等级系统，每次答题都能看到涨 |
| 所有权 | 宠物是自己的，给它起名字、换皮肤、布置小窝 |
| 成就感 | 每通关一个语法点获得"语法大师勋章" |
| 损失厌恶 | 连续登录奖励、连击 streak 断了会心疼 |
| 探索欲 | 隐藏关卡、彩蛋题、神秘宝箱 |

## 四、题型设计

### 4.1 "秒杀"模式 —— 单项选择
- 每题 10 秒倒计时（制造紧张感）
- 选对了：金币 +10，连击数 +1，屏幕闪绿光 + "Nice！"
- 选错了：不扣分，连击断，显示"差一点！下次注意~"
- 连击 5 次以上：额外金币 + 显示"🔥 on fire！"
- 连击 10 次以上：双倍金币

### 4.2 "极速拼装"模式 —— 词形转换
- 显示中文意思 + 乱序的词根和后缀碎片
- 拖拽正确的后缀拼出答案（如 `operate` + `tion` = `operation`）
- 拼对了碎片发光拼合，拼错了弹开震动

### 4.3 "语法手术"模式 —— 句型转换
- 显示一个句子 + 改造要求（变否定/变疑问/变被动）
- 句子像基因链条一样可点击修改
- 改好了 → 句子发光，"改造成功"
- 改错了 → 手术台冒烟，"不对不对，换一个"

### 4.4 "迷宫逃脱"模式 —— 完形填空 🏃
> 包装成"走迷宫"，每个空格是一扇锁住的门

- 显示一篇短文（80-120 词），中间有 5-7 个空位
- **类型 A：选词填空** — 右侧给出 10 个词，拖拽到正确空位
  - 类似上海中考的"选词填空"题型（从方框选词）
  - 给 10 个词，要填 6 个空，4 个干扰词
  - 每个词只能用一次
  - 拖对了 → 卡进空位发光，"咔嗒"一声锁住
  - 拖错了 → 词滑回原位，震动一下

- **类型 B：四选一完形** — 每个空 4 个选项
  - 类似传统完形填空，综合考语法 + 词汇 + 上下文逻辑
  - 选对 → 门打开，沿迷宫走一步
  - 选错 → 门撞上，退一步（视觉上退后）
  - 迷宫的背景是一条走廊，每答对一题小朋友的像素角色往前走一格

- 两种类型在关卡中随机混合出现
- **完成一篇** → "🏃 成功逃出迷宫！" + 宠物在出口等着鼓掌
- 覆盖中考完形填空的全部考点：上下文逻辑、连词、代词指代、动词时态、固定搭配

### 4.5 "Boss 战"模式 —— 单元测试
- 10 道混合题，Boss 有血条（10 格）
- 答对打掉一格，答错 Boss 反击（震动 + 掉血动画）
- 打倒 Boss → 获得该单元"语法大师"勋章

---

### 4.6 "词义射击"模式 —— 中英匹配 🔫
> 包装成"射击游戏"，匹配速度越快得分越高

- 屏幕上方显示一个中文词（如"冒险"），下方 4 个英文单词飞过
- 小朋友要在 **6 秒内**点中正确的英文（adventure）
- 正确：+8 金币，子弹命中特效 💥
- 错误：目标闪红，显示正确词
- 连击加分机制同选择题

**覆盖考点：** 中考 1600 词 · 中英词义对应

### 4.7 "单词拼写"模式 —— 看中拼英 ✍️
> 类似拼字游戏，但不是给字母乱序，而是补全缺失字母

- 显示中文释义 + 一个英文单词，但**少了几个字母**
- 例如显示："冒险" → a _ v _ n t _ r _
- 下方给出可点击的字母按钮（干扰项混入多余字母）
- 按顺序点击正确的字母补全单词
- 补对了：单词完整闪现发光，"拼写完美！"

**覆盖考点：** 中考核心词拼写 · 易错字母组合

### 4.8 "词组连连看"模式 —— 固定搭配 🔗
> 把两半"碎片"拼成正确的词组搭配

- 左列：动词/介词碎片（look / take / depend / listen...）
- 右列：搭配碎片（after / care / on / to...）
- 小朋友拖拽连线，把左右配对
- 例如：`look` ⟷ `after`，`take` ⟷ `care`，`depend` ⟷ `on`，`listen` ⟷ `to`
- 全部连对 → "搭配大师！" 动画效果像链条锁住

**覆盖考点：** 动词短语 · 介词搭配 · 中考常考固定搭配

### 4.9 "词义辨析"模式 —— 易混词对战 ⚔️
> 四个长得像/意思像的词，选对语境

- 给一个句子 + 空格，四个**易混词**作为选项
- 例如：
  ```
  This coat _____ me ¥300.
  A. spent   B. cost   C. paid   D. took
  ```
- 选对后弹出"辨析小贴士"（一句话讲清区别，不啰嗦）
- 答错 3 次同一组词 → 解锁该组词的"辨析卡片"（永久收藏在背包里）

**覆盖考点：** spend/cost/take/pay、few/a few/little/a little、sometime/sometimes/some time/some times 等中考高频易混词

### 4.10 "单词收集册"模式 —— 词汇本 📒
> 不是"做题"，而是"翻卡收集"

- 打开词汇森林地图，每个主题区域下有 10-20 个单词
- 每个单词初始为"暗卡"（背面朝上，只显示中文/首字母）
- 点击"翻卡" → 题目出现（选择题/拼写题/匹配题）
- 答对 → 翻卡成功，单词点亮，收入"单词收集册"
- 答错 → 卡翻回去，下次再战
- **收集完一个主题** → 获得"单词收集家"勋章 + 宠物获得主题装饰

**这个模式解决了背单词最核心的问题：** 不是一次把 20 个词全塞进去，而是一个一个"翻牌收集"，每次只聚焦一个词。

### 4.11 "侦探所"模式 —— 阅读理解 🔍
> 包装成"侦探破案"，每篇文章是一个"案件档案"，回答问题就是在找线索破案

**背景设定：**
小朋友扮演"语法侦探"，宠物是小助手。每篇文章是一个"案件档案"。答对所有问题 = 破案成功。

**文章选材（必须是 12 岁男生真正想读的）：**
| 类型 | 例子 | 篇幅 |
|---|---|---|
| 🦖 动物奇闻 | "会跳舞的蜜蜂"、"章鱼有三颗心脏" | 80-120 词 |
| 🚀 科技趣闻 | "为什么手机叫手机"、"游戏机进化史" | 100-150 词 |
| 🏀 体育故事 | "姚明小时候"、"梅西的足球" | 100-150 词 |
| 👻 趣味冷知识 | "为什么猫会咕噜"、"打哈欠会传染" | 80-120 词 |
| 🎮 游戏/动漫 | "宝可梦名字的由来" | 100-150 词 |
| 📖 小故事/寓言 | 简单的英文小故事，有情节反转 | 120-180 词 |

**三种出题方式（与上海中考一致）：**

**类型 A：四选一阅读（中考标准选择题）**
- 一篇文章 + 3-5 道选择题
- 覆盖：主旨大意、细节理解、推理判断、词义猜测
- 界面：左边/上方是文章，右边/下方是题目
- 选对 → "线索匹配！" 证据链图标亮一格
- 全对 → "🔍 案件告破！" + 金币奖励 + 侦探经验

**类型 B：阅读判断正误（中考 T/F 题型）**
- 一篇文章 + 4-5 个句子，判断 T (True) / F (False)
- 句子不是直接抄原文的，需要理解后判断
- 界面：文章上方显示，句子以卡片形式排列，每个卡片有 T/F 两个按钮
- 答对 → 卡片上盖绿色"✓"章
- 答错 → 盖红色"✗"章，显示正确答案

**类型 C：阅读回答问题（中考简答题型）**
- 一篇文章 + 3-4 个问题，需要**点击选择**答案
  - 注意：不手写，而是给出 3-4 个选项选一个（降低门槛，适配手机操作）
- 问题覆盖：What/Why/How/Where 等

**难度分级：**
| 星级 | 词数 | 题目数 | 时间 |
|---|---|---|---|
| ⭐ | ~80 词 | 3 题 | 3 分钟 |
| ⭐⭐ | ~120 词 | 4 题 | 4 分钟 |
| ⭐⭐⭐ | ~160 词 | 5 题 | 5 分钟 |

**关键体验设计：**
- 文章可以**点击查词**（点一下弹出中文释义，不收金币）—— 降低挫败感
- 每篇文章有"趣味结论"彩蛋，答完才显示（"原来章鱼真的有三颗心脏！"）
- **第一次解锁"阅读高塔"** → 宠物戴上侦探帽（新装饰）

### 4.12 "电台破译"模式 —— 听力理解 📻
> 包装成"电台情报员"，录音是收到的密电，回答问题就是在破译情报

**实现方式：** 用 edge-tts 免费生成 MP3 录音文件，浏览器播放，不需要任何硬件。

**背景设定：**
小朋友是"王牌情报员"，宠物是电台助手。每段录音是一条"加密情报"，答对问题 = 成功破译。

**五种听力题型（与上海中考一致）：**

| 题型 | 界面 | 时长 | 中考对应 |
|---|---|---|---|
| 🖼️ **听句选图** | 听一句描述 → 3 张图选 1 张 | ~10 秒 | 听力第一题 |
| 💬 **短对话问答** | 听两个人对话 → 选正确答案 | ~20 秒 | 听力第二题 |
| ✅ **短文判断** | 听一段短文 → 判断 4 个句子 T/F | ~40 秒 | 听力第三题 |
| ✍️ **短文填词** | 听短文 → 在屏幕空格里点选填词（不手写） | ~40 秒 | 听力第四题 |
| 🎙️ **长对话选择** | 听较长对话 → 回答 3-5 题 | ~60 秒 | 听力第五题 |

**界面设计：**
- 顶部：复古收音机 📻（旋钮、调频刻度、信号灯）
- 中间：播放控制条（▶ 播放 / ⏸ 暂停 / 🔁 重听 / 速度调节 0.75x/1x/1.25x）
- 底部：题目区域，听完才能看题（防止先看题再听）

**关键体验设计：**
- **每段录音只能听两遍**（模拟真实考场规则）
- **播放时题目不可见**（先听后看，训练真实听力能力）
- 听完后题目区域解锁，开始作答
- 全部答对 → "📻 情报破译成功！" + 金币奖励
- 答错三道以上 → "信号干扰……再来一次！"（允许重试，不惩罚）
- **调速功能**：0.75x 给基础弱的时候用，1.25x 给冲刺训练
- **第一次解锁"电波台"** → 宠物戴上耳机（新装饰）

## 五、⭐ 宠物系统（核心亮点）

### 5.1 定位
> 宠物是小朋友每天打开 App 的第一理由。"我想看看我的小火龙饿不饿。"

### 5.2 宠物基础设定

| 属性 | 说明 |
|---|---|
| 种类 | 初始可选：小火龙 🐉 / 小剑士 🗡️ / 小法杖 🪄 / 小幽灵 👻 |
| 名字 | 小朋友自己起名，默认为"小勇" |
| 等级 | 与用户等级同步（用户升级宠物也升级） |
| 外观 | 每 5 级进化一次形态（幼年 → 成长 → 成熟 → 完全体） |
| 心情 | 影响宠物的动作和对话框文字 |

### 5.3 宠物状态（4 项）

```
❤️ 饱腹度   ████████░░  80%  ← 满了会开心
😊 快乐度   ██████░░░░  60%  ← 满了会跳舞
⚡ 精力值   ████░░░░░░  40%  ← 低了会睡觉
✨ 干净度   ███████░░░  70%  ← 低了会脏兮兮
```

- 每项状态每天会自然衰减 10-20%
- 衰减到 0% → 宠物生病（不会死，但会不开心）
- 生病状态 → 答题金币收益减半（激励小朋友去照顾宠物）

### 5.4 金币消耗

| 项目 | 金币 | 效果 |
|---|---|---|
| 🍗 喂食 | 10 金币 | 饱腹度 +25% |
| 🎾 陪玩 | 15 金币 | 快乐度 +25% |
| 🛁 洗澡 | 5 金币 | 干净度 +30% |
| 💤 睡觉 | 免费 | 精力值回满（需等待 10 秒动画） |
| 🏠 买家具 | 50-200 金币 | 装饰宠物小窝 |
| 🎨 买皮肤 | 100 金币 | 改变宠物颜色/配饰 |
| 💎 进化石 | 500 金币 | 提前进化（达到等级才能用） |

### 5.5 宠物行为反馈

| 状态 | 宠物行为 |
|---|---|
| 饱腹 + 快乐 + 干净 ≥ 80% | 蹦蹦跳跳，冒爱心，偶尔说"主人好厉害！" |
| 任一状态 < 30% | 耷拉着脑袋，冒水滴/黑线，说"我饿了..." |
| 连续 3 天照顾得很好 | 触发稀有动作（翻跟头、喷小火苗） |
| 用户长时间没来 | 显示"你去哪了...我好想你" |

### 5.6 宠物进化路径

```
Lv.1-4  幼年体   →   小圆球模样，只会跳
Lv.5-9  成长体   →   长出四肢/翅膀，会跑
Lv.10-14 成熟体  →   威风凛凛，有专属动作
Lv.15+  完全体   →   终极形态，带特效光环
```

每个形态 2-3 种皮肤可解锁（金币购买）。

### 5.7 宠物 & 答题联动

- **今日第一次登录** → 宠物跳出来迎接，"主人你来啦！今天也要加油哦！"
- **连续答题 5 题正确** → 宠物撒花，"太棒了！"
- **Boss 战时** → 宠物在旁边加油助威，挥小旗子
- **打 Boss 赢了** → 宠物和主人一起庆祝，抱在一起转圈
- **每完成一个语法区域** → 宠物获得新装饰（小披风、小帽子）
- **⭐ 宠物学单词** → 词汇每收集 10 个新词，宠物会"学会"一个词，对话框里随机蹦出来用：
  - 学会 "adventure" 后 → 宠物会说："今天又是一场 great adventure！"
  - 学会 "delicious" 后 → 宠物说："这个食物好 delicious！"
  - 这种"双语混搭"特别符合 12 岁男生的说话习惯，他会觉得宠物很酷
- **完形填空逃脱成功** → 宠物在出口鼓掌"好厉害！出来了！"
- **阅读高塔爬到第 5 层** → 宠物戴上侦探帽（新装饰）

### 7.5 成就系统

### 7.6 ⭐ 智能复习系统（"复习训练营"）
> 不让孩子自己决定"今天复习什么"——系统自动安排。他只需要点"开始"。

**定位：** 每个错误都在合适的时间自动回来找他，不需要他自己翻错题本。

**核心机制：**
```
答题 → 错了 → 记录到复习队列 → 
1天后：第一次复习 → 对了 → 3天后第二次复习 → ...
                              → 又错了 → 重置回1天后
```

**复习调度算法（基于间隔重复）：**
| 答对次数 | 下次复习间隔 | 说明 |
|---|---|---|
| 第 1 次答对 | 1 天后 | 短期记忆转中期 |
| 第 2 次答对 | 3 天后 | 中期强化 |
| 第 3 次答对 | 7 天后 | 一周后巩固 |
| 第 4 次答对 | 14 天后 | 两周后深化 |
| 第 5 次答对 | 30 天后 | 长期记忆 |
| 第 6+ 次答对 | 标记"已掌握" | 不再出现（除非再次做错） |
| **任何一次答错** | **重置为 1 天后** | 重新循环 |

**触发方式（自动，不需要用户操作）：**
1. **每次答完一关/一篇** → 系统检查复习队列，如果有到期复习题 → 自动弹出一局"复习闪电战"（3-5 题）
2. **每天首次打开 App** → 如果有 5+ 道到期复习题 → 首页显示"⚠️ 有 X 道题待复习"
3. **"继续挑战"按钮** → 优先出到期的复习题，出完了才出新题

**奖励加成（激励复习）：**
- 复习题金币奖励 **1.5 倍**（答对 15 金币，新题只有 10 金币）
- 连续复习 5 天 → "复习连胜"勋章 + 宠物获得"小学霸"眼镜装饰
- 无视复习超过 3 天 → 宠物提示"主人，那些战利品还在等你呢……"

**跟战利品宝库的关系：**
- 战利品宝库是**手动查看** → 孩子可以翻卡片看历史错题
- 智能复习是**自动推送** → 到期了自动出现，不需要主动去找
- 两者同一套数据，只是入口不同

**覆盖范围：**
- 所有答错的语法题、词汇题、完形填空题
- 阅读理解答错的题
- 听力答错的题
- 单词收集册里答错 2 次以上的词
