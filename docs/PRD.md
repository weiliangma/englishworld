# EnglishWorld — 产品需求文档

> 目标用户：上海 12 岁男生（某人的儿子），初一/初二，有手机或平板，讨厌枯燥刷题
> 核心命题：**让语法练习像养宠物一样上瘾，而不是像做黄冈题库一样痛苦**

---

## 一、产品定位

### 一句话描述
> "一个做题赚金币养宠物的语法练习 App，只有你家孩子一个人用。"

### 核心原则

1. **5 秒规则** —— 任何页面如果 5 秒内没有让小朋友觉得"有点意思"，他就滑走了
2. **无痛学习** —— 把语法藏在玩法后面，不要让他觉得自己在做题
3. **即时反馈** —— 每个操作立刻有结果（对/错/金币/音效/动画）
4. **碎片化** —— 一局不超过 3-5 分钟，课间/蹲坑/等车都能来一局
5. **宠物驱动** —— 做题赚金币 → 金币养宠物 → 宠物开心了 → 还想再赚金币

## 二、用户画像

### 他（你家孩子）

| 属性 | 描述 |
|---|---|
| 年龄 | 12 岁，上海初一 |
| 英语现状 | 基础尚可，语法一团浆糊。知道"是这么说的"，但不知道为什么 |
| 痛点 | 老师讲语法太无聊，做题像坐牢。妈妈买的教辅连翻都不想翻 |

| 最大阻力 | 无聊。只要无聊，立刻放弃 |

### 一句话总结

> 他不会因为"为了你好"而打开 App。他打开它，是因为**想喂他的小恐龙**。

## 三、核心机制

### 3.1 世界观

**"语法世界"**

每个语法知识点 = 地图上的一块区域。小朋友扮演一个语法勇士，通过答题闯关，收集金币。金币用来**喂养和进化他的宠物**。

```
答题赚金币 → 养宠物 → 宠物进化 → 解锁新地图 → 答更多题 → 赚更多金币
                     ↑______________________________↓
```

### 3.2 驱动力设计（单人版）

| 驱动力 | 实现方式 |
|---|---|
| 好奇心 | 宠物下一阶段长什么样？新地图里有什么？ |
| 收集欲 | 勋章墙、宠物皮肤、称号 |
| 进步感 | 经验条 + 等级系统，每次答题都能看到涨 |
| 所有权 | 宠物是自己的，给它起名字、换皮肤、布置小窝 |
| 成就感 | 每通关一个语法点获得"语法大师勋章" |
| 损失厌恶 | 连续登陆奖励、连击 streak 断了会心疼 |
| 探索欲 | 隐藏关卡、彩蛋题、神秘宝箱 |

## 四、题型设计

### 4.1 "秒杀"模式 —— 单项选择
- 每题 10 秒倒计时（制造紧张感）
- 选对了：金币 +10，连击数 +1，屏幕闪绿光 + "Nice！"
- 选错了：不扣分，连击断，显示"差一点！下次注意~"
- 连击 5 次以上：额外金币 + 显示"🔥 on fire！"
- 连击 10 次以上：双倍金币

### 4.2 "极速拼装"模式 —— 词形转换
- 显示中文意思 + 乱序的词根和后缀碎片
- 拖拽正确的后缀拼出答案（如 `operate` + `tion` = `operation`）
- 拼对了碎片发光拼合，拼错了弹开震动

### 4.3 "语法手术"模式 —— 句型转换
- 显示一个句子 + 改造要求（变否定/变疑问/变被动）
- 句子像基因链条一样可点击修改
- 改好了 → 句子发光，"改造成功"
- 改错了 → 手术台冒烟，"不对不对，换一个"

### 4.4 "语法大乱斗"模式 —— 完形填空
- 一篇短文，空位占位
- 每填一空角色在战场上前进一步
- 全部填完 → "攻占了语法高地！"

### 4.5 "Boss 战"模式 —— 单元测试
- 10 道混合题，Boss 有血条（10 格）
- 答对打掉一格，答错 Boss 反击（震动 + 掉血动画）
- 打倒 Boss → 获得该单元"语法大师"勋章

## 五、⭐ 宠物系统（核心亮点）

### 5.1 定位
> 宠物是小朋友每天打开 App 的第一理由。"我想看看我的小火龙饿不饿。"

### 5.2 宠物基础设定

| 属性 | 说明 |
|---|---|
| 种类 | 初始可选：小火龙 🐉 / 小剑士 🗡️ / 小法杖 🪄 / 小幽灵 👻 |
| 名字 | 小朋友自己起名，默认为"小勇" |
| 等级 | 与用户等级同步（用户升级宠物也升级） |
| 外观 | 每 5 级进化一次形态（幼年 → 成长 → 成熟 → 完全体） |
| 心情 | 影响宠物的动作和对话框文字 |

### 5.3 宠物状态（4 项）

```
❤️ 饱腹度   ████████░░  80%  ← 满了会开心
😊 快乐度   ██████░░░░  60%  ← 满了会跳舞
⚡ 精力值   ████░░░░░░  40%  ← 低了会睡觉
✨ 干净度   ███████░░░  70%  ← 低了会脏兮兮
```

- 每项状态每天会自然衰减 10-20%
- 衰减到 0% → 宠物生病（不会死，但会不开心）
- 生病状态 → 答题金币收益减半（激励小朋友去照顾宠物）

### 5.4 金币消耗

| 项目 | 金币 | 效果 |
|---|---|---|
| 🍗 喂食 | 10 金币 | 饱腹度 +25% |
| 🎾 陪玩 | 15 金币 | 快乐度 +25% |
| 🛁 洗澡 | 5 金币 | 干净度 +30% |
| 💤 睡觉 | 免费 | 精力值回满（需等待 10 秒动画） |
| 🏠 买家具 | 50-200 金币 | 装饰宠物小窝 |
| 🎨 买皮肤 | 100 金币 | 改变宠物颜色/配饰 |
| 💎 进化石 | 500 金币 | 提前进化（达到等级才能用） |

### 5.5 宠物行为反馈

| 状态 | 宠物行为 |
|---|---|
| 饱腹 + 快乐 + 干净 ≥ 80% | 蹦蹦跳跳，冒爱心，偶尔说"主人好厉害！" |
| 任一状态 < 30% | 耷拉着脑袋，冒水滴/黑线，说"我饿了..." |
| 连续 3 天照顾得很好 | 触发稀有动作（翻跟头、喷小火苗） |
| 用户长时间没来 | 显示"你去哪了...我好想你" |

### 5.6 宠物进化路径

```
Lv.1-4  幼年体   →   小圆球模样，只会跳
Lv.5-9  成长体   →   长出四肢/翅膀，会跑
Lv.10-14 成熟体  →   威风凛凛，有专属动作
Lv.15+  完全体   →   终极形态，带特效光环
```

每个形态 2-3 种皮肤可解锁（金币购买）。

### 5.7 宠物 & 答题联动

- **今日第一次登录** → 宠物跳出来迎接，"主人你来啦！今天也要加油哦！"
- **连续答题 5 题正确** → 宠物撒花，"太棒了！"
- **Boss 战时** → 宠物在旁边加油助威，挥小旗子
- **打 Boss 赢了** → 宠物和主人一起庆祝，抱在一起转圈
- **每完成一个语法区域** → 宠物获得新装饰（小披风、小帽子）

## 七、功能模块（单人精简版）

### 7.1 首页
```
[宠物头像]     Lv.12 时态战士

经验条 ████████░░░░ 68%

今日完成：3 关    连击：🔥 12 天

[🚀 继续挑战]   [🏠 宠物小屋]   [🗺️ 地图]

宠物提示："主人，我饿了..."
```

### 7.2 语法地图
- 世界地图代替列表
- 每个语法点是一个"岛屿"：冠词森林、时态山脉、被动沼泽、从句深海
- 已完成 ✅  当前闪烁  未解锁 🔒
- 最终 Boss 城堡

### 7.3 宠物小屋（核心功能）
- 宠物展示 + 状态条
- 互动按钮（喂食/陪玩/洗澡/睡觉）
- 商店（买家具、皮肤、进化石）
- 宠物对话框（随机台词、卖萌）

### 7.4 战利品宝库（错题本）
- 不叫"错题本"
- 每道错题 = 一张"被击败的怪物卡片"
- 点击卡片 → 原题 + 解析 + "再来一战"
- 每周"怪物复活"：错题随机出现复习

### 7.5 成就系统
| 勋章 | 解锁条件 |
|---|---|
| 🏅 语法新兵 | 完成第一关 |
| 🗡️ 时态刺客 | 时态全三星通关 |
| 🛡️ 介词盾卫 | 介词正确率 90%+ |
| 🐉 屠龙者 | 击败任意 Boss |
| ⚡ 闪电手 | 连续 20 题 5 秒内答对 |
| 👑 全能王 | 所有语法点满星 |
| 🔥 不屈斗士 | 同一道题错 3 次后终于答对 |
| 🐾 最佳主人 | 连续 7 天照顾宠物 |

## 八、禁忌清单

| ❌ 不要 | 原因 |
|---|---|
| 不要任何社交/排行榜 | 这是给一个人的，不需要攀比 |
| 不要注册/登录/账号系统 | 本地存储即可，不需要网络 |
| 不要大段文字说明 | 12 岁男生不读超过 3 行的说明 |
| 不要"你做错了"冷冰冰的反馈 | 要有表情、动画、温度 |
| 不要纯打字输入 | 选择题/拖拽/点击为主 |
| 不要弹窗广告 | 这是给亲儿子的 |
| 不要强制每天学 1 小时 | 3-5 分钟一局，想打几局打几局 |
| 不要啰嗦的引导教程 | "开始挑战"一个按钮就够了 |
