# EnglishWorld — 中考英语练习平台 · 产品需求文档

> 目标用户：上海 12 岁男生，初一/初二，有手机或平板，讨厌枯燥刷题，喜欢打游戏、看 B 站、刷短视频。  
> 核心命题：**让语法练习像打游戏一样爽，而不是像做黄冈题库一样痛苦。**

---

## 一、产品定位

### 一句话描述
> "一个长得像游戏、用起来像闯关、最后真的能提分的中考英语语法练习平台。"

### 核心原则

1. **5 秒规则** —— 任何页面如果 5 秒内没有让小朋友觉得"有点意思"，他就滑走了
2. **无痛学习** —— 把语法藏在玩法后面，不要让他觉得自己在做题
3. **即时反馈** —— 每个操作立刻有结果（对/错/金币/音效/动画），绝不让他等着
4. **碎片化** —— 一局不超过 3-5 分钟，正好是蹲坑/等公交/课间操的时间
5. **羞耻感驱动** —— 弱化"你错了"，强化"你差 X 分就能超过某某"

## 二、用户画像

### 小明（虚构，但很真实）

| 属性 | 描述 |
|---|---|
| 年龄 | 12 岁，上海某公办初中初一 |
| 英语现状 | 基础尚可，语法一团浆糊。知道"是这么说的"，但不知道为什么 |
| 痛点 | 老师讲语法太无聊，做题像坐牢。妈妈买的教辅连翻都不想翻 |
| 娱乐偏好 | 王者荣耀（星耀）、B 站（半佛仙人/手工耿/各种游戏UP）、刷抖音 |
| 学习习惯 | 不做预习，不复习，考试前突击看笔记 |
| 心理特点 | 要面子（不想在同学面前丢人）、贪快（"这个我会了，下一题"）、怕麻烦 |
| 最大动力 | 排行榜里超过同桌 / 班级群里晒成绩 |
| 最大阻力 | 无聊。只要无聊，立刻放弃 |

### 人格画像总结

> 小明不会因为"这是为了你好"而打开这个 App。  
> 他打开它，是因为——**它看起来不像一个学习软件。**

## 三、产品核心机制

### 3.1 整体世界观

**"语法英雄"**（Grammar Heroes）

背景设定：语法世界被"混沌魔王"入侵，句子结构崩塌。学生扮演一个"语法勇士"，通过完成各种语法挑战，收集"语法碎片"，修复世界，最终挑战 BOSS。

- 每个语法知识点 = 一块地图关卡
- 每道题 = 一次战斗/挑战
- 正确 = 获得经验值 + 金币 + 连击加成
- 错误 = 掉血，但不会 game over（保护自尊心）

### 3.2 核心循环

```
登录 → 看一眼排行榜（被刺激） → 打一关（3-5分钟） → 
获得奖励 + 结算动画 → 看一眼新分数（满足） → 
发现下一关需要更高等级 → 再来一局
```

### 3.3 驱动力设计

| 驱动力 | 实现方式 |
|---|---|
| 使命/成就感 | 每通关一个语法点，获得"语法大师勋章"：时态战士、介词刺客、定语从句骑士... |
| 进步感 | 经验条 + 等级系统，每次答题都能看到经验值在涨 |
| 所有权 | 可自定义角色头像、称号、战斗特效颜色 |
| 稀缺性 | 每日限时挑战（每天只开放 3 次，错过就没了） |
| 社交比较 | 班级排行榜、好友 PK、本周进步榜 |
| 好奇心 | 隐藏关卡、彩蛋题、随机事件题（"神秘人发来一道超难题"） |
| 损失厌恶 | 连续登陆奖励（第 7 天送限定皮肤）、连击 streak 断了会心疼 |

## 四、题型设计（适配 12 岁男生心理）

### 4.1 "秒杀"模式 —— 单项选择 ✅
> 传统题型 + 游戏化包装

- 每题 10 秒倒计时（制造紧张感）
- 选对了：金币 +10，连击数 +1，屏幕闪绿光 + "Nice！"
- 选错了：不扣分，但是连击断了，显示"差一点！下次注意~"
- 连击 5 次以上：额外金币 + 显示"🔥 on fire！"
- 连击 10 次以上：触发"暴走"动画 + 双倍金币
- 包装风格：题目做成"怪物向你冲来，你必须选对技能才能击退它"

### 4.2 "极速拼装"模式 —— 词形转换 🧩
> 类似俄罗斯方块 + 拼词

- 屏幕上方显示中文意思，下方给出乱序的词根 + 后缀碎片
- 小朋友需要 **拖拽** 正确的后缀拼出正确答案
- 例如：显示"操作（名词）"，给出 `operate` + `tion` + `ing` + `ed`，拖 `tion` 拼成 `operation`
- 拼对了，碎片自己拼合成完整单词，闪一下
- 拼错了，碎片弹开 + 震动效果

### 4.3 "语法手术"模式 —— 句型转换 🔧
> 更像一个"句子改造游戏"

- 显示一个句子，底部给出改造要求（变否定 / 变疑问 / 变被动...）
- 页面以"手术台"风格呈现：句子是一排可点击的"基因链条"
- 小朋友点击需要修改的词，选择正确的新词替换
- 改好了 → 动画展示"改造成功"，新句子发光
- 改错了 → 手术台冒烟，"不对不对，换一个"
- 这是一个巧妙的简化：不需要手写整个句子，降低输入门槛

### 4.4 "语法大乱斗"模式 —— 完形填空 ⚔️
> 吃鸡/大逃杀的 UI 包装

- 页面显示一篇短文，空位用"？？？"占位
- 每空 4 个选项，选对才能继续
- 设计成"战场推进"：每填一个空，角色在战场上前进一步
- 全部填完 → "攻占了语法高地！"
- 额外奖励：如果连续填对 5 个空，触发"空降补给"——一道高难度附加题，对了给双倍经验

### 4.5 "Boss 战"模式 —— 综合测试 🐉
> 每个语法单元的大 Boss

- 10 道混合题型
- Boss 有血条（10 格），每题答对打掉一格
- 答错 Boss 会反击（屏幕抖动 + 角色掉血动画）
- 全部答完打倒 Boss → 获得该单元"语法大师"勋章
- Boss 设计：每个 Boss 有名字和立绘风格——
  - 时态龙（三头龙，分别代表过去/现在/将来）
  - 介词蛇（多足蜈蚣）
  - 被动蜘蛛（吐丝缠绕你）
  - 从句章鱼（多条触手）

## 五、视觉风格

### 5.1 整体风格
- **不要性冷淡风**（Notion/Tailwind 那种不行）  
- **不要幼稚**（幼儿卡通不行）  
- 要：**像素风 + 日系 RPG** 混搭  
  - 参考：星露谷物语 + 宝可梦 + 元气骑士
  
### 5.2 色彩
- 主色调：#2D5A27 墨绿（护眼、战斗感）
- 强调色：#FFD700 金色（金币、成就）
- 警示色：#FF4444 红色（Boss 攻击、错误）
- 成功色：#00FF88 翠绿（正确反馈）
- 背景：#1A1A2E 深蓝黑（深夜友好）

### 5.3 字体
- 英文：像素风格或圆润手写体（Comic Sans 的克制版）
- 中文：圆体/卡通风格（思源柔黑）

### 5.4 UI 元素
- 经验条：做成 RPG 经典的黄色条
- 金币动画：答题正确时金币从屏幕四周向中心飞入
- 角色：像素小人在页面角落，答题正确时跳跃/挥剑，错误时捂头
- 按钮：做成"宝箱"、"卷轴"、"盾牌"形状

## 六、功能模块

### 6.1 首页仪表盘
```
[角色头像] Lv.12 "时态战士"   [设置齿轮]

经验条 ████████░░░░ 68%

今日打卡：🔥 12 天连击

[继续挑战]    [每日任务]    [排行榜]

最新消息：你的朋友"王小明"刚刚超过了你的排名！
```

### 6.2 语法地图（核心导航）
- 用"世界地图"代替冰冷的列表
- 每个知识点是一个"岛屿"/"区域"
- 解锁顺序：简单 → 中等 → 困难
- 已完成区域显示旗帜 ✅
- 当前进度区域闪烁提示
- 区域名称（适配中二少年）：
  - "冠词森林"（冠词）
  - "时态山脉"（时态）
  - "被动沼泽"（被动语态）
  - "从句深海"（从句）
  - "介词迷宫"（介词）
  - "Boss 城堡"（综合测试）

### 6.3 排行榜
- 本班排行榜（接入班级功能）
- 好友排行榜
- 本周进步榜（给你那些总是最后几名的人一点希望）
- 神秘人榜（匿名的全国同年级学生，只显示昵称）

### 6.4 错题本（包装成"战利品收藏"）
- 不叫"错题本"，叫"战利品宝库"
- 每道错题以一个"被击败的怪物卡片"形式展示
- 点击卡片 → 显示原题 + 解析 + "再来一战"按钮
- 每周"怪物复活"：上周的错题随机出现在练习中，相当于复习

### 6.5 成就系统
勋章墙（中二命名）：
| 勋章 | 解锁条件 |
|---|---|
| 🏅 语法新兵 | 完成第一关 |
| 🗡️ 时态刺客 | 时态全三星通关 |
| 🛡️ 介词盾卫 | 介词正确率 90%+ |
| 🐉 屠龙者 | 击败任意 Boss |
| ⚡ 闪电手 | 连续 20 题 5 秒内答对 |
| 👑 全能王 | 所有语法点满星 |
| 🔥 不屈斗士 | 同一道题错 3 次后终于答对 |

### 6.6 个性化
- 角色皮肤解锁（像素剑士、忍者、魔法师、程序员...）
- 称号系统（时态大师、介词猎人、从句勇者...）
- 战斗特效（普通剑光、火焰、雷电、冰霜）
- 头像框（按等级解锁）

## 七、技术实现路线

### 阶段一：MVP（核心可用）
- 纯 Web 应用（PWA，可添加到手机桌面）
- 一个语法点（时态）的所有题型
- 答题 + 计分 + 反馈动画
- 基础的等级 + 经验条
- 不需要注册，本地 localStorage 存进度

### 阶段二：社区（社交化）
- 用户注册系统
- 班级 / 好友功能
- 排行榜
- 每日任务 + 连续打卡

### 阶段三：完整游戏化
- 全部语法点覆盖
- 成就系统
- 角色自定义
- Boss 战动画
- 隐藏彩蛋

### 阶段四：智能（AI 加持）
- 基于错题自动推荐薄弱点
- AI 生成变式题（同知识点不同包装）
- 个性化学习路径
- 中考真题预测

## 八、禁忌清单（不要做的）

| ❌ 不要 | 原因 |
|---|---|
| 不要大段文字说明 | 12 岁男生不读 >3 行的说明文字 |
| 不要注册后才可以用 | 必须点开就能玩 |
| 不要弹窗广告 | 家长会直接删掉 |
| 不要"你做错了，正确答案是 X" 这种冷冰冰的反馈 | 要有表情、动画、温度 |
| 不要长列表 | 任何超过 7 项的列表需要分页或折叠 |
| 不要"请家长扫码" | 这是给孩子的，不是给家长的 |
| 不要纯打字输入 | 手机/平板上打字体验差，选择题/拖拽/点击为主 |
| 不要强制每天学习 1 小时 | 3-5 分钟一局，想打几局打几局 |
| 不要啰嗦的引导教程 | "开始挑战" 一个按钮就够了 |
