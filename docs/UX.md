# EnglishWorld — 交互原型 & 用户旅程

> 小明的一天：一个 12 岁上海男生如何与 EnglishWorld 产生连接

---

## 一、用户旅程地图

### 典型场景：周一放学后

```
地铁上刷完 B 站 → 发现 EnglishWorld 通知
→ "你的同桌刚刚超过了你！"
→ 点开 → 10 分钟刷了 3 关
→ 追回排名 → 满足地关掉 → 第二天继续
```

### 完整旅程

```
发现阶段         首次体验          习惯养成           成瘾循环
  │                │                │                  │
  ▼                ▼                ▼                  ▼
同学推荐 /    注册（只需      每天打卡           打 Boss →
老师推荐 /    输入中二昵称）  领奖励 →           得勋章 →
应用商店         → 新手引导      做每日任务 →      发排行榜 →
               → 打第一个      看一眼排行榜 →     被同学超过 →
                Boss（时态）    被刺激再打一局      再打一局...
```

## 三、新手引导流程（3 步，绝不啰嗦）

> 12 岁男生的耐心：最多 3 步，每步一句话

```
Step 1: "请给你的勇士起个名字"
        [输入框] → 默认随机生成中二名字: "暗影剑客"
        按钮: [✨ 换一个]  [开始冒险]

Step 2: "选一个你最弱的语法点"
        展示 4 个语法点图标（不显示名称，只看 emoji）
        🌲 / ⛰️ / 🥦 / 🐙
        随便选一个 → 直接进入答题

Step 3: "看到题目选答案就行，别慌"
        第一题没有倒计时，没有连击显示
        答对后：🎉 "就这样，简单吧！"
        然后开启正常模式
```

**设计原则：**
- 零注册门槛：不需要手机号/邮箱
- 零教程：通过玩来学怎么用
- 零冷启动：第一次打开就能玩
- 第一次体验必须有"爽感"：第一题足够简单，让他觉得"我也可以"

## 四、支付心理（家长侧）

虽然孩子是使用者，家长是付费者，但：

**给孩子看的：**
- "解锁全部地图，成为语法英雄！"
- "月卡仅需 ¥19.9，不到一杯奶茶"

**给家长看的（分享页面）：**
- "覆盖上海中考全部语法考点"
- "已收录近 5 年 200+ 道真题"
- "每天 10 分钟，碎片化高效学习"
- "无广告，无社交风险，纯粹学习工具"

## 五、上线检查清单

### MVP 上线（10 分必须满足）

| # | 检查项 | 优先级 |
|---|---|---|
| 1 | 不用注册就能玩 | 必须 |
| 2 | 至少覆盖 2 个语法点（时态 + 冠词） | 必须 |
| 3 | 每题有反馈动画（正确/错误） | 必须 |
| 4 | 有金币和经验系统 | 必须 |
| 5 | 手机触摸体验流畅 | 必须 |
| 6 | 能本地存进度（下次打开不丢失） | 必须 |
| 7 | PWA 可添加到桌面 | 应有 |
| 8 | 排行榜（至少全服匿名版） | 应有 |
| 9 | Boss 战（至少 1 个 Boss） | 应有 |
| 10 | 错题本（战利品宝库） | 应有 |
