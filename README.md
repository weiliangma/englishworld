# EnglishWorld 🗡️

> 上海中考英语练习平台 — 让语法练习像打游戏一样爽

一个面向 12 岁上海男生的中考英语语法练习平台。语法知识点被包装成 RPG 世界地图，做题就是打怪升级打 Boss。

## 技术栈

- 前端：React + PWA（手机/平板友好）
- 后端：FastAPI + SQLite/PostgreSQL
- 设计：像素风 + 日系 RPG 混搭
